商业模式：部分开源 + 商业闭源

Community Edition

GitHub 开源 · BSL 1.1 · 纯 JS · 功能冻结

  • ~155 个 Actions（8 个工具）
  • Funnel 漏斗架构（Token 效率 4x）
  • 完整安全层（Token / 限速 / 超时 / CORS）
  • 事务回滚 + 组件名纠正
  • 预制体工作流 + 构建 + 编辑器内运行
  • 11 款 IDE 一键配置
  • 覆盖 80% 日常工作流
  • 冻结代码，只修 bug，不加新功能

目标：攒星 + 品牌 + 社区增长

Pro Edition

闭源 Rust .node · 全部工具重写 · 持续迭代

  • 社区版全部 actions 的 Rust 重写
  • + Pro 独占工具（动画/物理/脚本/原子宏）
  • + AI 场景生成器（自然语言 → 完整场景）
  • + 批量操作引擎（1 命令 = 50 次点击）
  • + 场景审计工具（5 秒完成 2 小时检查）
  • 升级后全部调用走 Rust，JS 不加载
  • 4 层保护: minify + 混淆 + 字节码 + Rust 二进制
  • 所有新功能只在 Rust 中开发

目标：收入 + 企业客户 + 持续迭代

Pro 独占功能：让用户心甘情愿付费

付费的核心不是"解锁更多工具"，而是"把 30 分钟的工作变成 30 秒"。

第一梯队：必杀级功能

🏗️
AI 场景生成器 P0
自然语言描述 → 自动创建完整场景结构。
"创建横版过关第一关：地面 + 3 个平台 + 2 个敌人巡逻点 + 终点旗帜 + 背景天空"
30 秒 vs 30 分钟
⚡
批量操作引擎 P0
一条命令批量修改多个节点/资源。
"把所有 enemy 节点的 scale 改为 1.5 并加上 RigidBody2D"
1 命令 = 50 次点击
🔍
场景审计工具 P0
自动检测场景问题 + 一键修复建议。
缺失 Layer / 无效引用 / 误操作 scale=0 / 碰撞层配置
5 秒 vs 2 小时检查

第二梯队：效率倍增器

📜
智能脚本脚手架 P1
生成代码 + 挂载组件 + 配置参数一体化。"给 Player 创建平台跳跃控制器，支持二段跳"→ 生成即可运行。
🎨
一键 UI 系统搭建 P1
自然语言 → 完整 UI 界面。"创建游戏主菜单：标题 + 开始/设置/退出按钮 + 版本号 + 背景" → 自适应布局。
🎬
动画工作流 P1
自然语言描述动画意图 → 自动创建。"UI 弹窗从 scale(0) 弹到 1.1 再回 1，0.3 秒 backOut" → 比手动快 10x。

第三梯队：企业级功能

📋
项目规范检查器 P2
自定义规则（命名规范 / 目录结构 / 节点限制）+ 自动检查 + 报告。团队协作刚需。
📹
操作日志与回放 P2
记录所有 MCP 操作 → 导出/回放/生成可复用模板。团队管理 + 可追溯。
🧠
Generative Action Engine P2
AI 自动生成新 action，无需等版本更新。独创能力，无替代品。

核心结论

社区版是"快照" — 155 actions 的 JS 实现，冻结在 GitHub 上攒星。
Pro 版是"未来" — 全部工具的 Rust 重写 + AI 智能工具，持续迭代。
升级 Pro 后全部调用走 Rust，JS 代码完全不加载。

社区版用户拥有所有的"砖头"（基础 actions），Pro 版提供"建筑师"（AI 智能工具）。
AI 智能工具通过 ExecutionPlan 调用任何底层 action，编排逻辑在 Rust 二进制中无法被抄袭。
