这是最适合直接落到你们当前插件上的方案。保留深色专业感，但把页面语气从“AI 控制器”改成“工程桥接控制台”。

  • 首页先给连接状态、工程名和快操作，不放提示词教学。
  • 把“控制”和“模块开关”合并，符合真实使用路径。

建议 可以作为主方案，再吸收 Quiet Dock 的低打扰细节。
