Concept 02

Project Pulse

这套更偏产品感和品牌感，适合你们如果想把 Aura 从“一个 MCP 面板”往“项目操作中枢”去表达。

  • 视觉更温和，降低赛博和模型感，更像成熟的创作工具。
  • 页面从项目、时间线、接入生态来组织，比纯功能分组更有叙事。
  • 适合做官网截图、商店页和宣传物料，辨识度明显高于普通深色面板。
优点 品牌气质最好，最不容易被看成“又一个 AI 插件”。
风险 如果实装时内容过多，暖色设计会更容易显拥挤。
建议 适合抽取其信息结构，再保留部分 Command Center 的硬朗按钮体系。
