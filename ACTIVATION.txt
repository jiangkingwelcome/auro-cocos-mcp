============================================
  Aura Pro — Activation Guide
============================================

Thank you for purchasing Aura Pro!

To activate your Pro license, follow ONE of these methods:


Method 1: Plugin Panel (Recommended)
-------------------------------------
1. Open Cocos Creator with the plugin installed
2. Open the plugin panel: Menu → 扩展 → Aura
3. Go to the "Settings" tab (设置)
4. Enter your License Key in the input field
5. Click "Activate" (激活)
6. Restart the plugin (Control tab → Restart)


Method 2: License File
-----------------------
1. Create a file named ".mcp-license" in the plugin directory:
     extensions/aura-for-cocos/.mcp-license
2. Write your License Key as the only content:
     COCOS-PRO-XXXXXXXX-XXXXXXXX-XXXXXXXX
3. Restart Cocos Creator


Method 3: Environment Variable
-------------------------------
Set the environment variable before launching Cocos Creator:

  Windows (PowerShell):
    $env:COCOS_MCP_LICENSE = "COCOS-PRO-XXXXXXXX-XXXXXXXX-XXXXXXXX"

  macOS / Linux:
    export COCOS_MCP_LICENSE="COCOS-PRO-XXXXXXXX-XXXXXXXX-XXXXXXXX"


Priority Order
--------------
If multiple methods are used, the priority is:
  Environment Variable > .mcp-license file > Panel setting


Verification
------------
After activation, the plugin panel "Settings" tab will show:
  "Pro Edition — 已激活 (Licensed to: xx...xx)"

All 230+ Pro actions and AI smart tools will be available.


Troubleshooting
---------------
- "License key signature verification failed"
  → Double-check the key for typos. Keys are case-sensitive.

- "License expired"
  → Contact the seller for a renewal key.

- Pro tools not appearing after activation
  → Restart the plugin: Control tab → Restart Service
  → Or restart Cocos Creator entirely.

- macOS: "cocos_pro.*.node cannot be opened"
  → Run: xattr -cr extensions/aura-for-cocos/
