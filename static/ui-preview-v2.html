4 种本质不同的 UI 方案：布局、交互、视觉隐喻完全不同。点击 Tab/导航 切换各页面。

浅色主题
仿 macOS 系统偏好设置，浅色背景，窗口标题栏带红绿灯，iOS 风格的分段控件 Tab。
适合追求「原生感」、不想让插件面板过于突兀的用户。

方案二：Terminal CLI
终端风
整个面板就是一个终端窗口，所有信息以命令行输出形式呈现，扫描线效果，光标闪烁。
适合硬核开发者，与 Cocos 的「开发工具」定位高度吻合。没有卡片、没有按钮，纯粹的文本流。
全终端 UI 扫描线特效 命令行输出格式 等宽字体

方案三：Bento Dashboard
数据仪表盘
数据优先的 Bento Grid 布局，状态页用大字号数字磁贴展示关键指标，彩色区分不同维度。
适合需要一眼看清全局状态的用户，信息密度高但不拥挤。
Bento 网格 大字号数字 彩色指标磁贴 信息密度优先
