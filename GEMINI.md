# Auro Cocos MCP

This is the source code for the **Aura for Cocos Creator** VS Code/editor plugin — a TypeScript/Node.js project that provides an MCP (Model Context Protocol) bridge between AI coding assistants and the Cocos Creator game engine editor.

## Project Type
- Language: TypeScript (Node.js)
- Framework: Cocos Creator Editor Extension
- Build: `npm run build` (tsc)

## Key Directories
- `src/` — TypeScript source code
- `dist/` — Compiled output (do not edit)
- `src/panels/` — Editor UI panel code
- `src/mcp/` — MCP server tools and handlers
- `src/routes/` — HTTP route handlers
